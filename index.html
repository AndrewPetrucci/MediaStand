How to Add Your Favorite Stuff

  1. Find a piece of media on any platform that supports embedding
  2. Usually embedding is an option under sharing and you can ask an LLM for a walkthrough if you are unsure
  3. Copy and paste the embedding code into whatever div you want. This should have an iframe element with a url.
  4. (optional) More comfortable users should incorporate a loop over an array of contentIds to render out content
